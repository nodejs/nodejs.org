### Notable changes

- **assert**:
  - backport some assert commits (Ruben Bridgewater) [#23223](https://github.com/nodejs/node/pull/23223)

  - added aborted property to request (Robert Nagy) [#20094](https://github.com/nodejs/node/pull/20094)
